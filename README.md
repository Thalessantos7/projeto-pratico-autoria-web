# 💻 Projeto Prático - Autoria Web

Este projeto foi desenvolvido para a disciplina de **Autoria Web** do curso de **Análise e Desenvolvimento de Sistemas** da **UNIFIP - Patos-PB**.
O objetivo é criar uma página utilizando apenas **HTML** e aplicar os conceitos aprendidos em aula, como estrutura básica, elementos semânticos, navegação interna e formulários.

---

## 🌐 Tema do site

O site apresenta uma **introdução ao mundo da programação**, abordando conceitos básicos, linguagens populares e dicas para quem deseja começar a programar.

---

## 🧱 Estrutura da página

A página foi desenvolvida utilizando a estrutura básica do HTML:

* `<!DOCTYPE html>`
* `html`
* `head`
* `body`
* `meta charset`
* `meta viewport`
* `title`

Também foram utilizados **elementos semânticos** para organizar o conteúdo:

* `header`
* `nav`
* `main`
* `section`
* `article`
* `aside`
* `footer`

---

## 🔗 Navegação interna

O site possui um **menu de navegação** com links que direcionam para diferentes seções da própria página utilizando âncoras (`#id`).

Exemplo:

```
<a href="#programacao">O que é programação?</a>
```

---

## 📚 Conteúdo da página

O site possui as seguintes seções:

* **O que é programação** – introdução ao conceito de programação
* **Linguagens populares** – apresentação de linguagens como Python, JavaScript e Java
* **Dicas para iniciantes** – sugestões para quem quer começar a programar
* **Curiosidades** – fatos interessantes sobre o mundo da tecnologia
* **Pesquisa sobre programação** – um formulário com perguntas sobre programação

---

## 📝 Formulário

A página possui um formulário com diferentes tipos de campos, incluindo:

* `input type="text"`
* `input type="email"`
* `input type="number"`
* `radio`
* `checkbox`
* `textarea`
* `button`
* `fieldset`
* `legend`
